The Maze Mirror


This project was making a hysical computing work with arduino and processing. We made a simple puzzle game using the characteristics of light reflection. The goal of this game is to make the laser light hit the laser sensor. As the stage goes by, the obstacles block the light and the player can use the mirrors to achieve the goal.


The obstacle comes up and blocks the laser light. And the user's goal is to make the light hit the specific sensor out of 5 sensors on the game board.


The obstacles come up according the stage description in the computer. And the user can use the mirror cube to change the path of the laser light. When the light hits the sensor, the red LED at the starting point turns on.
